< Project Overview >

This repository contains a Workout Website created exclusively for educational and portfolio purposes, to demonstrate frontend development skills and practical experience with Vue.js and Webpack module bundler.

< Description >

Features
    •    Two fully implemented and functional pages: Home and Pricing
    •    Built using Vue.js with a component-based architecture
    •    Routing implemented with Vue Router
    •    Fully responsive design
    •    Optimized for all screen sizes from 320px to 2500px
    •    Desktop, tablet, and mobile friendly
    •    Modern UI based on a Figma design template

Technologies Used
    •    HTML5
    •    CSS3
    •    JavaScript (ES6+)
    •    Vue.js
    •    Vue Router
    •    Webpack
    •    SASS (SCSS)

Webpack Configuration Includes:
    •    webpack
    •    webpack-cli
    •    webpack-dev-server
    •    vue-loader
    •    vue-template-compiler
    •    css-loader
    •    vue-style-loader
    •    sass-loader
    •    node-sass

Key capabilities:
    •    Development mode with hot module replacement (real-time updating)
    •    Automatic compilation of Vue single-file components (.vue)
    •    SASS preprocessing
    •    Asset bundling and optimization
    •    Separate dist output folder for production-ready builds

⚠️ Important Notice

The repository was created solely for educational, demonstration, and portfolio purposes.
